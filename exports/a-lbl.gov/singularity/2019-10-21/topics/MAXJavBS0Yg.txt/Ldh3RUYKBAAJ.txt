Call for papers:

We are pleased to introduce the 1st International Workshop on Containers=20
and New Orchestration Paradigms for Isolated Environments in HPC (CANOPIE=
=20
HPC), to be hosted at SC19: The International Conference for High=20
Performance Computing, Networking, Storage and Analysis. The objective of=
=20
this workshop is to serve as the principal venue for leaders in the field=
=20
to stimulate research and interactions in relation to cutting-edge=20
container technologies, virtualization, and OS system software as it=20
relates to supporting High Performance Computing (HPC). The first iteration=
=20
of this workshop will give special attention to work which can provide=20
real-world experiences to containerization in HPC, as well as promote the=
=20
use of containers and engage in a collaborative manner with the larger HPC=
=20
community.

Container computing has revolutionized the way groups are developing,=20
sharing, and running software and services.  This has initially been led by=
=20
growth with Docker, which has provided an ecosystem of tools to enable=20
container based computing.  This paradigm shift has since made in-roads in=
=20
the HPC community enabled by container runtimes like Singularity, Shifter,=
=20
and Charliecloud, which allow end-users to run containers in environments=
=20
where standard Docker tools would not be feasible.  Orchestration systems=
=20
such as Kubernetes allow for quick, scalable service deployments in=20
conjunction with HPC. While this adoption is growing, the larger HPC=20
community still has many questions around this new model. There still exist=
=20
several research questions related to containerization in HPC that the=20
community must address.

Topics of Interest (but are not limited to):

- HPC Container Runtimes
- Utilizing Containers to improve Portability and Reproducibility
- Experience using containers for HPC high-performance computing=20
applications including data analytics, machine learning, modeling and=20
simulation.
- Virtualization in supercomputing environments & HPC clusters
- Container runtime designs for Exascale
- HPC in the cloud
- Accelerator and GPU virtualization
- Container security and trust models including container distribution
- In-situ HPC & Analysis coupling experiments
- Workflow ensemble coupling with containers
- Performance and scaling studies with containers and/or VMs
- Container services orchestration and microservices with HPC
- On-node resource partitioning with OS mechanisms
- Combined container and OS/VM designs
- DevOps models with containers and image registries
