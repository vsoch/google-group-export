Greetings Singularity community!=20

It is my very great pleasure to announce the release of Singularity 3.0.0!=
=20
 Singularity 3.0 represents a new milestone in Singularity development. The=
=20
source code has been completely re-written from the ground up and is now=20
implemented entirely in C and Go.  This huge effort will simplify and=20
facilitate future development and features.

But no need to wait on new features.  Singularity 3.0 is packed with new=20
goodies!=20

With this release Sylabs unveils a new suite of cloud tools to expand the=
=20
Singularity ecosystem: The Container Library, Cloud Builder, and Keystore

https://cloud.sylabs.io/library

You can use these alpha-stage services to create, store, distribute, and=20
sign/verify/trust your containers!

Singularity 3.0 also comes with a new set of network management features.=
=20
 It is now possible to carry out complex network configuration within your=
=20
container.

And there is even more!  Singularity 3.0 has a whole slew of new privilege=
=20
options allowing admins to have fine-grained control over the permissions=
=20
that containers need to run.  3.0 offers cgroups support, and enhanced=20
instance support, including the ability to =E2=80=9Cboot=E2=80=9D a contain=
er and start an=20
init process within the container. It also expands Singularity support for=
=20
Docker and OCI by allowing users to run local images that are saved in=20
these formats.=20

For a complete list of the new changes in Singularity 3.0. See the=20
CHANGELOG.md

https://github.com/sylabs/singularity/blob/master/CHANGELOG.md

To download Singularity 3.0, follow the link below.

https://github.com/sylabs/singularity/releases/download/v3.0.0/singularity-=
v3.0.0.tar.gz

As always, please report any bugs or issues here:

https://github.com/sylabs/singularity/issues/new

Thanks!  And happy containerizing!=20

Dave
